# PulseID Access Badge System

## Overview

The PulseID Access Badge System is a system for managing access to a building using NFC badges.

## Requirements

- Python 3.10+
- PC/SC reader
- NFC badges
- Linux/MacOS

## Features

- Register cards
- Log auth events
- Generate PDF reports
- Web UI
- API

## Installation

1. Clone the repository
2. Install the dependencies
3. Run the application

```bash
pip install -r requirements.txt
```

```bash
python pulseid.py --server
```
